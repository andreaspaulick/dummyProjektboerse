Usage:

List posts: http://localhost:8045/posts

List a single posts: http://localhost:8045/posts/<id>

List the post-titles only: http://localhost:8045/posts/titles

Add post: http://localhost:8045/posts/add?title=<title>&content=<content>&status=publish

Delete post: http://localhost:8045/posts/delete?id=<id>

Delete all post: http://localhost:8045/posts/deleteall

 

Add Post via JSON-REST-Request: http://localhost:8045/posts/jsonadd

Expected JSON Format:

{
   "title": "<title>",
   "content": "<content>",
   "status": "publish"
}
