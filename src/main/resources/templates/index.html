Usage:

(GET) List posts: http://localhost:8045/posts

(GET) List a single post: http://localhost:8045/posts/id/<id>

(GET) List the post-titles only: http://localhost:8045/posts/titles

(POST) Add post: http://localhost:8045/posts/add?title=<title>&content=<content>&status=publish

(GET) Delete post: http://localhost:8045/posts/delete?id=<id>

(DELETE) Delete all posts: http://localhost:8045/posts/deleteall

 

(POST) Add Post via JSON-REST-Request: http://localhost:8045/posts/jsonadd

Expected JSON Format:

{
   "title": "<title>",
   "content": "<content>",
   "status": "publish"
}
